Quick Start
Get started with AIA system in minutes with this basic setup guide. 
# 1. Access the main platform
https://013a.tech

# 2. Launch local development
cd /frontend
npm install && npm start

# 3. Open immersive interface
open aia-main-frontend.html

# 4. Use keyboard shortcuts
1-4: Switch views
Space: Auto-rotate
R: Reset view

Agent Coordination
Example of coordinating multiple agents for a complex task. 
// Multi-agent task coordination
const task = {
  objective: "Deploy enhanced frontend",
  agents: [
    "cryptography-agent",
    "three-js-ui-optimizer",
    "gcp-deployment-orchestrator"
  ],
  strategy: "collaborative-sprint",
  security: "enterprise-grade"
};

orchestrator.executeTask(task);
